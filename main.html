Book Library

My Java project is a Book Library system integrated with an SQL database. Customers can buy books, employees can manage the book collection by performing CRUD (Create, Read, Update, Delete) operations. I used design patterns like the Factory and Builder patterns, respecting OOP principles and LISKOV. These design principles facilitate efficient data manipulation and ensure a user-friendly experience within the application.
